# 20 | 怎么比较工作区和暂存区所含文件的差异？

`git diff`: 默认查看的是工作区和暂存区所含文件的差异

`git diff -- <filename>` : 查看工作区与暂存区指定<文件>的差异

​	例如： `git diff -- readme.md`
